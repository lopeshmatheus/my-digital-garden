Salvar as modificações atuais em outro local e resetar o projeto descartando todos os commits 

### Uso 
#### git stash list
Lista as stashs salvas
#### git stash~={red} nome=~
Salva uma stash com nome
#### git stash apply ID
Recupera a stash
#### git stash show
#### git stash clear
Remove todas as stashes
#### git stash drop ==nome==
Remove uma stash
